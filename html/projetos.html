Projetos com Arduino

Contador

Este projeto apresenta um contador de 1 a 9 feito com Arduino, mostrando de forma simples como controlar saídas digitais e criar sequências numéricas. Ele é ideal para quem está começando na eletrônica, pois demonstra princípios básicos de programação, temporização e lógica. A partir desse contador, o projeto pode evoluir para versões mais avançadas, como controle por botões, uso de múltiplos displays, contagem regressiva, integração com sensores ou até comunicação com celular e computador, tornando-se a base para sistemas mais completos de automação.

Semáforo

Este projeto utiliza o Arduino para controlar dois semáforos, simulando o funcionamento real de um cruzamento entre duas vias. Ele demonstra conceitos importantes de temporização, lógica de controle e coordenação entre dispositivos, sendo ideal para aprender como sistemas de tráfego funcionam. A partir dessa base, o projeto pode evoluir para versões mais complexas, como sensores que detectam veículos, ajustes automáticos no tempo do sinal, integração com pedestres, comunicação entre vários cruzamentos ou até sistemas inteligentes de controle de tráfego.

Semáforo Físico

Este projeto físico mostra a aplicação prática do Arduino no controle de componentes reais, permitindo visualizar o funcionamento do sistema fora do ambiente virtual. Ele demonstra como eletrônica, programação e montagem se unem para criar um protótipo funcional, servindo como base para projetos maiores e mais complexos.

Led acendendo com Potenciômetro

Desenvolvi um projeto com Arduino utilizando um potenciômetro para controlar três LEDs (código). Conforme o valor lido aumenta, os LEDs acendem de forma sequencial: primeiro o LED 1, depois o LED 2 e, por fim, o LED 3. Esse sistema permite visualizar facilmente a variação do sinal analógico, funcionando como um indicador de nível simples e eficiente.

Contador com Potenciômetro

Projeto de Contador com Potenciômetro (código), onde o valor analógico do potenciômetro é convertido em etapas para acionar diferentes estados no Arduino. Conforme o usuário gira o potenciômetro, o contador avança ou retrocede, permitindo controlar funções de forma simples e intuitiva.
