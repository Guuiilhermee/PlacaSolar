O que é Arduino

É uma plataforma de hardware e software de código aberto, projetada para tornar a eletrônica acessível para todos, permitindo a criação de projetos interativos e protótipos

O que é um microcontrolador

É um pequeno computador em um único chip que contém um processador, memória e periféricos de entrada/saída (E/S) para controlar tarefas específicas em sistemas embarcados, como em eletrodomésticos e controles remotos

Microcontrolador do Arduino: modelo e fabricante

Atmel, ATmega328p UNO

Qual a linguagem de programação do Arduino

C++ e Micropython

Onde o Arduino é programado (IDE)

Arduino IDE

Qual é o valor em volts da entrada analógica no Arduino e a resolução em bits

0-5V 0-1023

Qual o valor da entrada/saída digital em volts no Arduino

0 é 0V e 1 é 5V

O que é um multímetro

Instrumento de medição portátil usado para medir grandezas elétricas como tensão (voltagem), corrente (amperagem) e resistência (ohms)

O que é escala de medição

É um sistema de categorização e quantificação de dados que determina como as variáveis são avaliadas e analisadas

Qual a lei que o multímetro aplica para realizar as medidas

Lei de Ohms

O que é teste de continuidade e para que serve

É um procedimento para verificar se um circuito elétrico está fechado e se a corrente pode fluir entre dois pontos

Como é a medição de uma resistência no resistor

Use um multímetro na escala de resistência (Ω), conectando as ponteiras em suas extremidades

Como é a medição de uma tensão no circuito com resistor e led

Usar um multímetro na função de voltímetro, configurado para medir tensão contínua (VCC), e conectar as pontas de prova em paralelo com o componente

Como é a medição de uma corrente elétrica no circuito

A medição da corrente elétrica pode ser feita em série com um circuito usando um multímetro ou indiretamente com um alicate amperímetro, que não exige interrupção do circuito
